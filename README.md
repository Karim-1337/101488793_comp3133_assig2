# COMP 3133 Assignment 2

101488793_comp3133_assig2 

## Stack

- **Frontend:** Angular (Material + Bootstrap), GraphQL (Apollo), Axios for file upload  
- **Backend:** Node.js, Apollo Server, MongoDB (Mongoose)  
- **Docker:** `docker compose` runs MongoDB, API, and SPA

## Run (Docker)

```bash
docker compose up --build
```

- App: http://localhost:4200  
- GraphQL: http://localhost:5000/graphql  
- MongoDB Compass: `mongodb://localhost:27018` (database `comp3133_assignment2`)
